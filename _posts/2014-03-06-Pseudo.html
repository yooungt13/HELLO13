阴影效果利用到了两个伪类：:before和:after

分别设置伪类的css，通过z-index置于父div下，达到角阴影的效果。
这里注意：若想控制子div的z-index在父div之上，要将父div的position设为relative，子div设为absolute。即便父div不设置z-index的值，将默认为0，子div可通过z-index值正负控制为父div上或下。 {% highlight css %} #box::before, #box::after{ position: absolute; content: ''; background: #fff; width: 40%; height: 94%; top: 15px; box-shadow: 0px 8px 20px #000; z-index: -1; } #box::before{ left: 10px; transform:rotate(-2deg); -webkit-transform:rotate(-2deg); -moz-transform:rotate(-2deg); -o-transform:rotate(-2deg); } #box::after{ left: 58%; transform:rotate(2deg); -webkit-transform:rotate(2deg); -moz-transform:rotate(2deg); -o-transform:rotate(2deg); } {% endhighlight %}
